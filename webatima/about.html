About

This is a web user interface for the Catima library, a C++ library for calculation of energy loss, range, angular scattering and time of flight of the particle passing through a matter. The library is based on a physics used in the ATIMA code, however its not a 100% copy of the ATIMA physics. For the details about the calculation check Helmut's ATIMA webpage. Manual

Pojectile

1. Select the projectile either by writing the nucleus symbol, ie 12C, or by setting A, Z fields
2. Select the kinetic energy, velocity etc
3. Mass of the projectile is calculated from the AME16.

Target Material

Material layer can be added by clicking Add Matter button. Material must be specified either by symbol field or by Z. If atomic number of material is omited or 0, the atomic weight will be used. Selected compund materials are supported, Z and symbopl names are compatible with MOCADI compund materials. Material can be added from already saved file by clicking on Add From File button. In this case the material as added to the existing list.

To edit existing material layer click the edit button or double click on the corresponding material row.

Calculate

The results of the calculation can be obtained via Calculate button.

Save/Load

Current configuration can be saved via menu item Save. The selected projectile and the material layers are save in JSON text format. The saved file can be loaded via menu item Load
